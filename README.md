# Ecommerce Store

A full stack ecommerce web application where users can browse products, add them to cart, and make secure payments.
This project demonstrates full stack development using modern web technologies.

---

 Features

* Product listing page
* Add to cart functionality
* Secure online payment
* Responsive user interface
* Backend API for product and payment handling

---

 Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Payment Integration: Stripe

---

 Installation

1. Clone the repository

git clone https://github.com/kavyayaa/-ecommerce--store-.git

2. Install dependencies

npm install

3. Run backend server

cd backend
npm start

4. Run frontend

npm start

---

 Environment Variables

Create a `.env` file inside the **backend** folder and add:

STRIPE_SECRET_KEY=your_stripe_secret_key
MONGO_URI=your_mongodb_connection_string
